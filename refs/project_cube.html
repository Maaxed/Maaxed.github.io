Project Cube

2016 - 2019

Objectifs

Project Cube est un jeu 3D que j'ai créé pour apprendre le fonctionnement de OpenGL et du langage c++. Je n'utilise pas de game engine et uniquement des bibliothèques graphiques et mthématiques telles que SDL et SOIL.

Afain de dévelloper plus efficacement le jeu, je l'ai recréé en C#, un language plus facile à deboger offrant des fonctionnalités qui réduisent le code nécessaire. La seul bibliorthèque utilisée est OpenTK qui permet d'intéragir avec OpenGL.

Contenu du Jeu

Le jeu présente un monde générée de manière procédurale. Celui-ci contient des montagnes, des rivières et des arbres formés uniquement à partir de cubes. Le joueur peut se déplacer dans ce monde (presque) infini.

Algorithmes utilisés

  • Perlin noise
    Pour générer le monde
  • Ray tracing
    Une version simplifiée de l'algorithme de ray tracing pour gérer la colision entre le joueur et le monde. Il est également utilisé pour la génération des arbres.
  • Octree
    Une structure de données sous forme d'arbre utilisée pour stocker les cubes du monde.
